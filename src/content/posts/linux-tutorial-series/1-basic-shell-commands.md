## 练习题

### 实践题

请尽量让每一小题都使用一条命令完成：

1. 在当前 Shell 中输出 Linux 内核版本信息。
2. 在当前 Shell 中输出当前系统的发行版名称和版本信息。
3. 列出根目录 `/` 下的目录和文件。
4. 输出当前系统时间。
5. 查看当前系统已安装的显卡驱动信息，并给出显卡型号、驱动版本和驱动支持的 CUDA 版本。
6. 查看当前已安装的 CUDA Toolkit 版本。

### 思考题

1. Linux 内核版本和 Linux 发行版版本有什么区别？
2. 根目录 `/` 下有很多一级目录，例如 `/home`、`/data` 等。它们分别是什么，通常承担什么作用？
3. Linux 的文件组织与管理方式，和 Windows 系列系统有哪些主要区别？
4. 实践题第 5 问中看到的 CUDA 版本，与第 6 问中看到的 CUDA Toolkit 版本，有什么区别？
5. 是否可以让多个版本的 CUDA Toolkit 共存，并在需要时完成版本切换？如果可以，通常如何实现？
